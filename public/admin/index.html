Yang’s Log CMS

Free visual editing for posts, built around Decap CMS and tuned for a GitHub Pages workflow.

Open Writing Studio Back to site

Recommended workflow

  1. Run npm run dev locally.
  2. In another terminal, run npm run cms:proxy.
  3. Open http://localhost:4321/admin/ to edit posts visually.
  4. For advanced MDX components or bilingual fine-tuning, continue using the built-in Studio page or your code editor.

Why local only?

This site is published on GitHub Pages with a fully free stack. Decap CMS is therefore configured for free local visual editing instead of a hosted OAuth flow that would require extra infrastructure.
